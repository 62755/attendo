# attendo

## Description
Student attendance management web app built with Vue.js and Supabase.
Developed to efficiently track and manage student attendance during exams.

## Features & Overview
- Teachers can create exam sessions, select a classroom, and assign a supervisor.
- Studentsâ€™ attendance is marked by teachers during the session.
- Attendance records are stored in the backend (Supabase) and can be reviewed later.
- Simple and intuitive user interface for managing classes, exam sessions, and supervisors.

## Technologies Used
- Vue.js for the frontend
- Supabase for authentication, database, and backend services
- Tailwind CSS for styling
